# Metric Master: AI Performance Lab

An interactive educational tool to explore Confusion Matrices, Specificity, Recall, F1-Score, and Multi-class classification strategies.

## Features

- **The AI Doctor (Binary Classification):**
  - Interactive sliders to adjust True Positives, False Negatives, etc.
  - Real-time visualization of patient population.
  - Live calculation of Specificity, Precision, Recall, and F1-Score.
  
- **The Fruit Sorter (Multi-class Classification):**
  - Editable 3x3 Confusion Matrix.
  - Breakdown of "One-vs-Rest" metrics for each class.
  - Macro-Average F1 calculation.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.
